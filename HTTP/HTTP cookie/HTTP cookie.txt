 * Ketika Web Browser melakukan request selanjutnya, maka Web Browser akan menyisipkan cookie yang sudah
   diterima di request sebelumnya
 * Dari cookie tersebut, Server bisa mengetahui apakah request tersebut merupakan client request yang sudah
